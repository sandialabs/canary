.. _canary-vvtest:

VVTest test scripts
===================

``canary`` can discover, interpret, and execute tests written in the ``vvtest`` format. This makes it possible to run existing vvtest suites under ``canary`` while using ``canary`` features such as filtering, reporting, and (when applicable) scheduled/HPC execution.
